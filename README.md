# 🦺 Worker Safety

Sistema desenvolvido para auxiliar no monitoramento e gestão da segurança dos trabalhadores em ambientes industriais, empresariais e operacionais.

## 📋 Sobre o Projeto

O Worker Safety é uma plataforma voltada para a prevenção de acidentes e o acompanhamento das condições de segurança no ambiente de trabalho. O sistema permite registrar informações, monitorar indicadores e facilitar a comunicação entre trabalhadores, gestores e equipes de segurança.

## 🎯 Objetivos

- Promover a segurança dos trabalhadores.
- Reduzir riscos e acidentes de trabalho.
- Centralizar informações relacionadas à segurança ocupacional.
- Facilitar o acompanhamento de ocorrências e inspeções.
- Gerar relatórios para análise e tomada de decisão.

## 🚀 Funcionalidades

- Cadastro de trabalhadores.
- Registro de ocorrências e incidentes.
- Controle de Equipamentos de Proteção Individual (EPIs).
- Monitoramento de riscos.
- Relatórios de segurança.
- Dashboard com indicadores.
- Gestão de inspeções.
- Histórico de ocorrências.
